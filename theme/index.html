USC Subsite Framework is a collection of CSS and JQuery files built with responsiveness in mind from which all USC sites in future can be built.See dumentation for use. The Framework is built on the Zurb Foundation Responsive Framework but is built in such a way that it can support other Responsive grids as well

Key elements of the framework are:

Standardised Color Pallete

Created with the mobile first philiosophy

Support for video backgrounds in embedded

Consistent header and footer realizations

Off canvas menu

Light footprint resulting in fast page loads

Developmer are free to choose for the USC color palette but MUST consume styling classes as defined by the pallette for branding realization consistency
